# lBook

可以自定义获取网页小说的扩展。

## 安装方式

1. 打开 VS Code
2. 进入扩展市场
3. 搜索 `lBook`
4. 点击安装

## 扩展设置

### `lbook.view`
- 类型: string
- 默认值: "text"
- 描述: 书籍打开方式，默认 `text`，可选输出面板 `output`

## 快捷键

列出扩展自定义的快捷键及其功能。
| 快捷键 | 命令 | 功能 |
|--------|------|------|
| `alt+left` | `lbook.textPrevious` | 在书籍章节打开时切换到上一章 |
| `alt+right` | `lbook.TextNext` | 在书籍章节打开时切换到下一章 |
